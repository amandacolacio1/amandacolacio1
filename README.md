# 👩‍💻👋 Olá, me chamo Amanda! 

Sou **Desenvolvedora Full Stack** atuando no mercado de tecnologia desde **2021**, com experiência no desenvolvimento de aplicações **web**, **mobile** e **automações**, utilizando tecnologias como **JavaScript, TypeScript, Python, React, React Native, Next.js e Node.js**.

Sou formada em **Sistemas de Informação** pela **PUC Minas** (2023) e pós-graduada em **Desenvolvimento Full Stack** pela mesma instituição. Atualmente moro em **Contagem, Minas Gerais**, na região metropolitana de **Belo Horizonte**.

Este GitHub é o meu espaço para compartilhar conhecimento, acompanhar minha evolução e registrar minha trajetória como desenvolvedora. Aqui você encontrará projetos desenvolvidos para estudar novas tecnologias, colocar conceitos em prática e explorar diferentes soluções para problemas do dia a dia.

Acredito no aprendizado contínuo e utilizo este portfólio como uma forma de consolidar conhecimentos, experimentar novas ferramentas e documentar minha evolução profissional ao longo da carreira.
